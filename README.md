# Tetris en consola - C++
## Descripción

Este proyecto es un minijuego de tetris desarrollado con C++ para consola de texto. Mi objetivo es aplicar mis previos conocimientos de programación con C++ y aprender en el camino.

## Características del juego

- ↔️ **Movimientos laterales** - La pieza se puede mover a los lados
- 🔃 **Rotaciones** - Las piezas pueden rotar
- 🔥 **Dificultad** - La dificultad es progresiva de acuerdo a los puntos
- ‼️ **Combos** - Hay alertas de dependiendo el combo de líneas completas 

## Instalación
Compilar el proyecto:
```
g++ -Iinclude main.cpp src/*.cpp -o tetris.exe
```
